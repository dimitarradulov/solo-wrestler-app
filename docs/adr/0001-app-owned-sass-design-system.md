# App-Owned SASS Design System

Solo Wrestler uses an app-owned SASS design system instead of external UI or CSS frameworks such as Tailwind or Bootstrap. SASS emits CSS custom properties as the public styling contract so Ionic components can consume the same tokens through their runtime variables, while Google Fonts may be loaded directly for the selected brand typefaces. The MVP is dark-only, matching the scoreboard training aesthetic and avoiding a second unused theme surface.
